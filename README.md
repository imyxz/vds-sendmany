# vds-sendmany

这是一个用于vds币批量转账的工具，理论上衍生于bitcoin并提供与bitcoin一致的rpc接口的币都可以使用

* [x] 基本的转账功能
* [] 支持非本地钱包
* [] 地址本
* [] 交易查询

> Nuxt + Electron

## Build Setup

``` bash
# install dependencies
$ npm install # Or yarn install

# development with vue devtools
$ npm run dev

# build for production
$ npm run build
